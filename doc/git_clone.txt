git clone [<options>] <repository> [<directory>]

<repository>: URL or path of the repository to clone.

<directory>: Name of the directory to create for the cloned repository.

Options:

    --bare: Clone as a bare repository.
    --depth <depth>: Limit the number of commits fetched.
    --branch <branch>: Clone a specific branch.
    --recurse-submodules: Clone submodules.
